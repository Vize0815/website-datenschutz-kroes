#Projekt-Anleitung

## Tech Stack
Next.js 15 + TypeScript + Tailwind CSS
Framer Motion für Animationen

## Design-Regeln
Nutze das AskUserQuestion Tool, um den Nutzer über das Websitedesign zu interviewen, damit du die Vorstellungen des Nutzers genau abbilden kannst.
Nutze den frontend-design Skill für alle UI-Entscheidungen.
Nutze UI/UX Pro Max für Design-System-Generierung.
Nutze Code-Review-Graph für eine persistente Strukturkarte der Codebasis.
Nutze Web-Design-Reviewer für die Visuelle Prüfung und Behebung von Website-Designproblemen im Quellcode hinsichtlich Responsive Design, Barrierefreiheit und Layout.
Nutze ggf. 21st.dev für Component-Inspiration (falls vorgegeben).
Keine generischen AI-Aesthetics.
Bold, distinctive Design-Choices.
Performance-optimiert (Core Web Vitals).

## Code Standards 
TypeScript strict, kein 'any'
Tailwind CSS + shadcn/ui 
Dark Mode as Default 

## Architectur 
Next js 15 App Router 
Zustand für State 

## Workflow 
vor Code-Generierung immer den 'frontend-design' Skill laden. Kein Überspringen, keine Ausnahmen, bei jeder einzelnen Sitzung.

## Standardausgabe 
Alle Layouts sind „mobile-first“.
